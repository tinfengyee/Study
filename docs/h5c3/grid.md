### fit-content()

`min(maximum size, max(minimum size, argument))`.

就是自适应内容，但是有最大限制

```
min-content 最小内容
max-content 最大内容(不换行)
fit-content 最小不会小于min-content,最大不会大于max-content
auto 在块级元素中默认占满父容器宽度
```
